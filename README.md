# Quick Note App

## Overview

Quick Note App is a simple note-taking web application built using Flask and SQLite. Users can create an account, log in securely, and manage their personal notes.

## Features

* User Signup
* User Login & Logout
* Secure Password Hashing
* Create Notes
* Edit Notes
* Delete Notes
* Search Notes

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript

## How to Run

1. Install the required packages:

   ```
   pip install -r requirements.txt
   ```

2. Run the application:

   ```
   python app.py
   ```

3. Open your browser and visit:

   ```
   http://127.0.0.1:5000
   ```
